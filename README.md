## 🏃 시작 가이드

### 요구 사항
<!-- 누군가 clone해서 실행하려고 할 때 필요한 요구사항 및 버전 기재 -->
1. Node.js
2. Npm 
<br>

### 설치
```
$ git clone https://github.com/likelion-LIKIT/Front-end
$ cd Front-end
```

### 실행
```
$ npm install
$ npm start 
```

## ✨ 주요 기능

1. 동아리 주요 일정 등록 및 조회
2. 기수 별 개발 서비스 등록 및 조회 
3. 전체 공지 및 과제 공지 등록 및 조회

<br/>

## ⚙️ 개발 컨벤션
1. Airbnb javascript 코드 컨벤션 준수
2. Angular commit message 컨벤션 준수
3. file 상단에 작성자 이름 기재
4. env file 변경 시 통보
5. 함수(메서드) 당 하나의 기능만 구현
6. inline style 자제

<br/>

## 🗂️ 디렉터리명 
- common: 전역으로 사용할 디렉터리 관리
- pages: 페이지 파일 관리
- components: 컴포넌트 파일 관리
- hooks: hook 파일 관리
- styles: css 파일 관리
- types: ts + react 조합의 파일 관리
- assets: 이미지 및 아이콘 등 정적 파일 관리
- utils: 유틸 함수 관리
- api: api 목록 관리
- constant: 정적 데이터 관리
- stores: 전역 상태 파일 관리
- lib: 기타
